Algo

2019-02-16 Written by krejac
#privacy  #vpn 

So I just tried out AlgoVPN by Trail of Bits. It worked a lot better than both my personal VPN and my coporate VPN. Trail of Bits describes Algo as follows:

“Algo VPN is a set of Ansible scripts that simplify the setup of a personal IPSEC and Wireguard VPN. It uses the most secure defaults available, works with common cloud providers, and does not require client software on most devices.”

I deployed it to a (now destroyed) cheap Digital Ocean instance - and it was an easy config with WireGuard on both macOS and iOS.

Also it was much faster than my other VPNs. Highly recomended.
